# gestion-computadores

# 1. Descripcion del proyecto

Sistema para la gestion de computadores ya sea de escritorio o portatil.


# 2.Tecnologias usadas

.-   **Laravel** – Framework principal para el backend y la estructura MVC.
    
-   **Blade** – Motor de plantillas para las vistas.
    
-   **PHP** – Lenguaje base del proyecto.
    
-   **MySQL** – Base de datos relacional para almacenar los registros.
    
-   **Bootstrap 5** – Framework CSS para la interfaz y componentes responsivos.
    
-   **Laravel Breeze / Auth** – Sistema de autenticación para gestionar el login.
